# Frequently Asked Questions

## General Questions

??? question "What is PortMaster?"
    PortMaster is a free, open-source tool that helps you install game ports on Linux-based handheld devices. It provides a simple interface to download and manage ports for your device.

??? question "Is PortMaster free?"
    Yes, PortMaster is completely free and open-source. You can use it without any cost.

??? question "Which devices does PortMaster support?"
    PortMaster supports a wide range of handheld devices running custom firmware. See our [Supported Handhelds](installation/supported-handhelds.md) page for the complete list.

??? question "How do I install PortMaster?"
    See our [Installing PortMaster](installation/installing-portmaster.md) guide for step-by-step instructions.

---

## About Ports

??? question "What's the difference between 'Ready to Run' and commercial ports?"
    **Ready to Run Ports:** Free, open-source, or officially licensed games that install completely from PortMaster. No additional files needed.
    
    **Commercial Ports:** Ports that require you to own the game. You must provide your own legally purchased game files.

??? question "How do I install a port?"
    See our [Installing Ports](installation/installing-ports.md) guide for complete instructions.

??? question "Where do I get game files for commercial ports?"
    You must purchase games from legitimate sources like Steam, GOG, Epic Games Store, or Itch.io. After installing a port from PortMaster, visit the [Games](games.md) page, find your game, and click Details for file placement instructions.

??? question "Can I install ports without WiFi?"
    Yes! Download the port's `.zip` file on a computer and place it in the autoinstall folder. See [Installing Ports - Option 2](installation/installing-ports.md#option-2-offline-installation).

## Troubleshooting

??? question "PortMaster won't launch"
    - Verify you have an active internet connection
    - Make sure PortMaster is installed correctly
    - Update your custom firmware to the latest version
    - Try reinstalling using the [installation guide](installation/installing-portmaster.md)

??? question "A port won't run after installation"
    - Verify game files are in the correct location (check the port's Details page)

    - Check the `log.txt` file in the port's folder for error details
    - See our [Runtimes Guide](installation/runtimes.md) if runtimes are missing

??? question "Port doesn't appear after installation"
    - Refresh your games list (method varies by custom firmware)
    - Restart your device
    - Make sure you exited PortMaster properly

??? question "PortMaster update failed"
    - Check your internet connection
    - Make sure you have enough storage space
    - Try downloading the latest installer manually and reinstalling

## Technical Questions

??? question "What are runtimes?"
    Runtimes are software libraries that ports need to run. They include game engines, frameworks, and dependencies. PortMaster installs these automatically when needed, or you can pre-install them for offline use. See the [Runtimes Guide](installation/runtimes.md).

??? question "Can I use PortMaster on stock firmware?"
    PortMaster is designed for custom firmware (CFW) like ROCKNIX, ArkOS, muOS, and KNULLI. Stock firmware is generally not supported.

??? question "How much storage do I need?"
    This varies by port. Most ports are under 500MB, but some larger games can be several gigabytes. Check individual port details before installing.

??? question "Can I delete ports I don't want?"
    Yes! Use the **Manage Ports** section in PortMaster to uninstall any port.

## Our Stance on Piracy

??? question "What is your stance on Piracy"
    At PortMaster, we believe in supporting game developers and publishers by ensuring that all game ports are created and used with legitimately purchased software.
    
    **How PortMaster Works:**
    
    Our platform enables you to install ports from games you've bought on trusted platforms like Steam, GOG, Epic Games Store, and Itch.io. This approach ensures that developers receive the compensation they deserve, encouraging further innovation and quality in the gaming industry.
    
    **Ready to Run Ports:**
    
    Some ports in PortMaster are "Ready to Run" - these include:
    
    - Open-source games that are freely distributable
    - Games where developers have personally given us permission to distribute
    - Freeware titles with proper licensing
    
    These ports install completely and require no additional files.
    
    **Commercial Ports:**
    
    All other ports require you to provide your own legally obtained game files from legitimate purchases.
    
    **What We Require:**
    
    - All ports require legally obtained game files (unless Ready to Run)
    - Purchase games from legitimate sources
    - Respect developer intellectual property
    
    **What We Don't Support:**
    
    - Unlicensed or pirated game files
    - Unauthorized distribution of copyrighted content  
    - Pre-packaged ports that include commercial game files without permission
    - Any activity that violates intellectual property rights
    
    **Zero Tolerance:** PortMaster has zero tolerance for piracy. We are free and open-source software, but we never distribute copyrighted game files or enable piracy.
