PortMaster is your all-in-one manager for native game ports on Linux-based handhelds. Designed for retro and indie gaming enthusiasts, it simplifies downloading, installing, and managing PC game ports, ensuring smooth performance and compatibility across a variety of devices and firmware.

## Key Features

- Browse and install from a library of over 1000 native game ports.
- Cancel downloads anytime to save bandwidth.
- Preview games before installing to make informed choices.
- Filter and sort ports by genre, runtime, updates, and installation status.
- Manage installed games with ease — install, update, or uninstall.
- Automatic handling of dependencies and runtimes.
- Customizable themes for a personalized interface.
- Lightweight, fast, and optimized for handheld performance.
- Compatible with major Linux-based custom firmware platforms.

## Installation

#### Prerequisites
Your device must run a supported Linux-based custom firmware, such as AmberELEC, ArkOS, or muOS.

#### Steps
Most firmware includes PortMaster by default. If not, follow the manual installation guide on the [:material-download: Installation](installation/installing-portmaster.md) page.
